# Specify Prompt

## User Story: Мягкое удаление пользователя через API

**Как пользователь системы, я хочу иметь возможность мягко удалить пользователя через API, чтобы ограничить его доступ в систему.**

### Acceptance Criteria:
1. **Запрос на удаление пользователя** должен помечать пользователя, найденного по `email` удаленным.
2. **Ответ API** должен содержать статус код 204 без дополнительной информации.
3. **Найти или обновить информацию пользователя** после мягкого удаления должно быть невозможно.
4. **В случае ошибки**, например, если `email` не найден, API должен вернуть соответствующий статус-код с сообщением об ошибке.
5**Мягкое удаление** должно помечать пользователя как «удален», но не удалять физически данные о нем.

### UI:
- Без UI для фичи, это API-эндпоинт.
